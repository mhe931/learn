Advanced Data Modeling

Slowly Changing Dimensions (SCD)

Handling history in data warehousing is critical. Here are the standard types:

  • Type 0: Retain Original. No changes allowed.
  • Type 1: Overwrite. No history kept. Good for error correction.
  • Type 2: Add New Row. The gold standard for tracking history. Uses `EffectiveDate`, `EndDate`, and `IsCurrent` flags.
  • Type 3: Add New Column. partial history (e.g., `PreviousCity`).
  • Type 4: History Table. Separate current state from history.
  • Type 6: Hybrid (1+2+3). Powerful but complex.
